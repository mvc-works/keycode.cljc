
Keycode library
----

### Usage

[![Clojars Project](https://img.shields.io/clojars/v/mvc-works/keycode.svg)](https://clojars.org/mvc-works/keycode)

```edn
[mvc-works/keycode "0.1.2"]
```

```clojure
keycode.core/back-space
keycode.core/tab
keycode.core/enter
keycode.core/escape
keycode.core/space
keycode.core/left
keycode.core/up
keycode.core/right
keycode.core/down
keycode.core/n0
keycode.core/n1
; ...
keycode.core/a
keycode.core/b
keycode.core/c
; ...
; ... and more
```
